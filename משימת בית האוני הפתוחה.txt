משימת בית האוני הפתוחה
מטלת בית – מערכת לניהול סטודנטים

Backend – ASP.NET Core/framework Web API

דרישות:  
- לבנות API RESTful לניהול טבלת סטודנטים, כולל CRUD מלא.  
- גישה לנתונים, שימוש ב- Entity Framework Core או כל דרך אחרת שתבחרו.  
- טיפול בשגיאות HTTP סטנדרטיות (404, 400, 500 וכו’).  
- שמירה על מבנה פרויקט תקני עם הפרדה בין Controllers, Services ו- Models.  

שדות הטבלה:  
StudentId ,FullName ,BirthDate ,Average grade ,IsActive


פעולות נדרשות:  

- GET /students

- GET /students/{id}

- POST /students

- PUT /students/{id}

- DELETE /students/{id}

בסיס הנתונים:  

יש להשתמש באחת מהאפשרויות: SQLite או In-Memory Database או כל מסד נתונים אחר  
שתבחרו (העיקר שההרצה תהיה בלי צורך להתקין DB בזמן בדיקת המטלה 😉).
